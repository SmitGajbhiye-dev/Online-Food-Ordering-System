2. Click **+** → **New repository**
3. Name: `Online-Food-Ordering-System`
4. Description: "Complete Online Food Ordering System - DBMS & Web Development Project"
5. Select **Public**
6. Click **Create repository**

### **Step 2: Copy HTTPS URL**
- Click green **Code** button
- Copy HTTPS URL: `https://github.com/YOUR_USERNAME/Online-Food-Ordering-System.git`

### **Step 3: Connect Local Repository**
```bash

```

### **Step 4: Rename README**
```bash

git add README.md
```
