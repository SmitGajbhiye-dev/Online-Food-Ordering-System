## ✅ Quick Checklist

### Step 1: Create GitHub Repository
```
1. Go to github.com
2. Click + (top right) → New repository
3. Name: Online-Food-Ordering-System
4. Description: Complete Online Food Ordering System - DBMS Project
5. Select: Public
6. Create repository (do NOT initialize)
```

### Step 2: Copy Repository URL
```
Click green "Code" button → Copy HTTPS URL
Example: https://github.com/YOUR_USERNAME/Online-Food-Ordering-System.git
```

### Step 3: Connect Local Repository
```bash
# In PowerShell in your project folder, run:
git remote add origin https://github.com/YOUR_USERNAME/Online-Food-Ordering-System.git
```

### Step 4: Rename README File
```bash
# Rename GitHub_README.md to README.md
Rename-Item GitHub_README.md README.md
```

### Step 5: Push to GitHub
```bash
# Stage, commit, and push
git add README.md
git commit -m "Set GitHub README"
git branch -M main
git push -u origin main
```
